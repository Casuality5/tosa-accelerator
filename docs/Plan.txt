WHAT WE ARE BUILDING
----------------------------------------------------------------

A Python program that:
- Takes an image as input
- Takes a task as input (one of 14 tasks)
- Outputs the most suitable object in that image for that task

Example:
    input  -> image of a kitchen, task = "serve wine"
    output -> "wine glass"

Runs fully on CPU. No GPU needed. No hardware. No FPGA.
That is all Stage 2A requires.

----------------------------------------------------------------
THE 14 TASKS (these are fixed, from the COCO-Tasks paper)
----------------------------------------------------------------

1.  step on something
2.  sit comfortably
3.  place flowers
4.  get potatoes out of fire
5.  water plant
6.  get lemon out of tea
7.  dig hole
8.  open bottle of beer
9.  open parcel
10. serve wine
11. pour sugar
12. smear butter
13. extinguish fire
14. pound carpet

----------------------------------------------------------------
HOW THE PROGRAM WORKS (the pipeline)
----------------------------------------------------------------

STEP 1 - Object Detection
    Use OWL-ViT (zero shot object detector from Google/HuggingFace)
    Give it the image + a list of candidate object names for the task
    It finds and draws bounding boxes around those objects
    Unlike YOLO it is not limited to 80 classes
    It can find anything you describe in text

STEP 2 - Scoring with CLIP
    For each detected object crop that region from the image
    Feed the cropped image + task description to CLIP
    CLIP gives a similarity score - how well does this object
    match what is needed for the task
    CLIP understands visual context not just label names

STEP 3 - Pick best
    Object with highest CLIP score wins
    Print the label and bounding box

LIBRARIES NEEDED (install these first)
----------------------------------------------------------------

    pip install torch torchvision
    pip install transformers
    pip install Pillow
    pip install openai-clip

OR install all at once:
    pip install torch torchvision transformers Pillow openai-clip

Models that will auto-download on first run:
    - OWL-ViT  (google/owlvit-base-patch32)  ~600 MB
    - CLIP     (ViT-B/32)                    ~350 MB

Download these on a good internet connection before sitting
down to code so you are not waiting during development.

To pre-download run this once:
    python -c "from transformers import pipeline; pipeline('zero-shot-object-detection', model='google/owlvit-base-patch32')"
    python -c "import clip; clip.load('ViT-B/32', device='cpu')"
